Using scrollTop

Get percentage


function update(e) {
  let html = document.scrollingElement;
  let height = html.scrollHeight - html.offsetHeight;
  let percentScrolled = html.scrollTop / height;
}

Apply values


scrollProgress.scale.set(1, percentScrolled);

Lerp

Interpolate between two points


function lerp(start, end, t) {
	return (1 - t) * start + t * end;
}

Can be used for momentum
